Seo and core web vitals in website

  1. FID

    2. FID (First Input Delay) ek SEO Core Web Vitals metric hai jo website ki responsiveness ko measure karta hai, specifically, jab pehli dafa user kisi action ko perform karta hai aur website ka response time kitna hai.

  2. LCP

    3. LCP (Largest Contentful Paint) teesra SEO Core Web Vitals metric hai jo website ke loading speed ko evaluate karta hai, specifically, largest content element ka time measure karta hai jo screen par appear hota hai.

  3. CLS

    4. CLS (Cumulative Layout Shift) ek aur SEO metric hai jo website ke layout ki stability ko measure karta hai, yani agar page load hota hai aur elements unexpectedly move karte hain, toh kitna displacement hota hai.
